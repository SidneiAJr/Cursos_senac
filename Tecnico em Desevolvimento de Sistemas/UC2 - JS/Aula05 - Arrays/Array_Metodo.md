# Metodos dos Arrays:

Métodos dos Arrays:

* push(): Adiciona um ou mais elementos ao final de um array e retorna o novo comprimento do array.

* pop(): Remove o último elemento de um array e o retorna.

* length: Retorna o número de elementos em um array ou o comprimento de uma string.

* toLowerCase(): Converte todos os caracteres de uma string para minúsculas.

* toUpperCase(): Converte todos os caracteres de uma string para maiúsculas.

* trim(): Remove os espaços em branco do início e do final de uma string.

* includes(): Verifica se o array (ou string) contém um determinado valor. Retorna true ou false.

*  replace(): Substitui a primeira ocorrência de um conjunto de caracteres por outro. (Note que replaceAll() pode ser utilizado para substituir todas as ocorrências, dependendo do ambiente).

*  unshift(): Adiciona um ou mais elementos ao início de um array e retorna o novo comprimento do array.

*  shift(): Remove o primeiro elemento de um array e o retorna.

*  concat(): Concatena (une) dois ou mais arrays (ou valores) em um novo array.

*  sort(): Ordena os elementos de um array.
