# Git | Comandos CLI (Linha de Comando)

Antes de utilizar o Git, é essencial conhecer alguns comandos básicos do terminal (CLI - Command Line Interface).

---

## 📂 Navegação entre Diretórios

### 🔹 `cd` — Change Directory

Usado para entrar em uma pasta.

```bash
cd nome-da-pasta
```

Voltar um nível:

```bash
cd ..
```

Ir para a raiz (Linux/Mac):

```bash
cd /
```

---

## 📄 Listar Arquivos

### 🔹 `ls -l`

Lista arquivos em formato detalhado (permissões, tamanho, data).

```bash
ls -l
```

---

### 🔹 `ls -a`

Mostra todos os arquivos, incluindo arquivos ocultos (ex: `.git`).

```bash
ls -a
```

---

## 📁 Criar Diretório

### 🔹 `mkdir` — Make Directory

Cria uma nova pasta.

```bash
mkdir nome-da-pasta
```

---

## 💻 Comandos Mais Usados no Dia a Dia

```bash
cd nome-da-pasta
cd ..
ls -l
ls -a
mkdir nova-pasta
```

---

## 🧠 Dica Importante

Esses comandos são fundamentais para organizar arquivos antes de iniciar um projeto com Git.

Exemplo de fluxo inicial:

```bash
mkdir meu-projeto
cd meu-projeto
git init
```
